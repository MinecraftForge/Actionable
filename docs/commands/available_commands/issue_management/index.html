Issue Management Commands

move​

Moves the issue this command is run in to another repository.

Aliases: transfer
Syntax: /move <repo>

Parameters:

  • repo (GHRepository): the repository to move the issue to

Requirements: can_manage_issue, in_pull_request

title​

Set the title of the issue.

Syntax: /title <title>

Parameters:

  • title (String): the new title of the issue

Requirements: can_manage_issue

lock​

Lock the issue.

Syntax: /lock [reason]

Parameters:

  • reason (LockReason): the reason for locking the issue

Requirements: can_manage_issue

clock​

Lock and close the issue.

Syntax: /clock [reason]

Parameters:

  • reason (LockReason): the reason for locking the issue. If resolved, the close reason will be completed, otherwise it will be not_planned

Requirements: can_manage_issue
